File upload

Due to the limitations of CSS you currently need to set the filename by hand. This snippet of JavaScript should do the trick. We plan to utilize native file picker CSS at some point. 

document.querySelectorAll('input[type=file]').forEach(elem => elem.addEventListener('change', (e) => {
    const id = e.currentTarget.getAttribute('id');
    const element = document.querySelector(`.cosmo-picker__name[for=${e.currentTarget.getAttribute('id')}]`);
    element.textContent = e.currentTarget.value;
}))

<div class="cosmo-input__group">
    <label for="filePicker" class="cosmo-label">Pick file</label>
    <div class="cosmo-input cosmo-input--picker">
        <label class="cosmo-picker__name" for="filePicker">Filename</label>
        <label class="cosmo-picker__button" for="filePicker">
            <span class="mdi mdi-upload mdi-24px"></span>
        </label>
        <input style="display: none" type="file" id="filePicker">
    </div>
</div>

Buttons

Cosmo CSS consists of only one type of button. Currently, there are no plans to add additional styles 

<button class="cosmo-button">Button</button>

Progress

The HTML 5 progress tag can be used to display a progress bar on the web. Currently, only determinate progress bars are supported by Cosmo CSS. If you wish to use an indeterminate progress bar you can fallback to the default styling of the browsers. 

<progress class="cosmo-progress-bar" value="13" max="100"></progress>
