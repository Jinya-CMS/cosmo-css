HTML controls

Cosmo provides a well crafted design for your form controls, see the demos below.

Form input

<div class="cosmo-input__group">
    <label for="textInput" class="cosmo-label">Text input</label>
    <input type="text" id="textInput" class="cosmo-input">
    <label for="select" class="cosmo-label">Select</label>
    <select id="select" class="cosmo-select">
        <option value="1" selected>Option 1</option>
        <option value="2">Option 2</option>
        <option value="3">Option 3</option>
        <option value="4">Option 4</option>
    </select>
    <div class="cosmo-checkbox__group">
        <input type="checkbox" id="checkboxInput" class="cosmo-checkbox">
        <label for="checkboxInput">Checkbox input</label>
    </div>
    <label class="cosmo-label cosmo-label--radio">Radios</label>
    <div class="cosmo-radio__group">
        <input name="radio" type="radio" id="radioInput1" class="cosmo-radio">
        <label for="radioInput1">Radio input 1</label>
        <input name="radio" type="radio" id="radioInput2" class="cosmo-radio">
        <label for="radioInput2">Radio input 2</label>
        <input name="radio" type="radio" id="radioInput3" class="cosmo-radio">
        <label for="radioInput3">Radio input 3</label>
    </div>
</div>

File upload

Cosmo CSS supports custom styling for file inputs, which makes it super easy by only applying the cosmo-input class to the file input. 

<div class="cosmo-input__group">
    <label for="filePicker" class="cosmo-label">Pick file</label>
                <input type="file" class="cosmo-input">
</div>

Buttons

Cosmo CSS consists of only two types of buttons. The first type is a classical button just like you know it from your operating system or other frameworks. 

<button class="cosmo-button">Button</button>

Circular button

The other button type are circular buttons, they come in three sizes, small, medium and large. Medium is the default size and doesn't require any extra classes. They are best combined with an icon, just as seen in the example below. 

<button class="cosmo-circular-button cosmo-circular-button--small">
    <span class="mdi mdi-theme-light-dark"></span>
</button>
<button class="cosmo-circular-button">
    <span class="mdi mdi-theme-light-dark"></span>
</button>
<button class="cosmo-circular-button cosmo-circular-button--large">
    <span class="mdi mdi-theme-light-dark"></span>
</button>

Progress

The HTML 5 progress tag can be used to display a progress bar on the web. Currently, only determinate progress bars are supported by Cosmo CSS. 

<progress class="cosmo-progress-bar" value="13" max="100"></progress>

Indeterminate state

As of version 0.12.0 Cosmo supports the indeterminate state of progress bars.

<progress class="cosmo-progress-bar"></progress>

Messages

Cosmo CSS provides fur types of messages to display information to your user, information, warning, positive and negative

Information

I am just an information, don't worry about me

Warning

I am a warning message, keep your eyes open and check before you click

Positive

I am a positive message, something worked or is a good thing to do

Negative

I am a negative message, something didn't work or is dangerous to do 

<div class="cosmo-message cosmo-message--positive">
    <span class="cosmo-message__header">Positive</span>
    <p class="cosmo-message__message">
        This worked very well
    </p>
</div>
<div class="cosmo-message cosmo-message--negative">
    <span class="cosmo-message__header">Negative</span>
    <p class="cosmo-message__message">
        This didn't work well
    </p>
</div>
<div class="cosmo-message cosmo-message--information">
    <span class="cosmo-message__header">Info</span>
    <p class="cosmo-message__message">
        Just an information
    </p>
</div>
