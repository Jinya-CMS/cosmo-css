HTML controls

Cosmo provides a well crafted design for your form controls, see the demos below.

File upload

Native browser styling

Since version 0.12.0 there is a class available for the native browser styling, which eradicates the need for several elements and JavaScript. Due to browser limitations the look and feel is different. 

<div class="cosmo-input__group">
    <label for="filePicker" class="cosmo-label">Pick file</label>
                <input type="file" class="cosmo-input">
</div>

Custom classes

CSS only allows a bit of styling for file pickers. If you want the button to have an icon and the button on the end of the input, you should take the code below. 

document.querySelectorAll('input[type=file]').forEach(elem => elem.addEventListener('change', (e) => {
    const id = e.currentTarget.getAttribute('id');
    const element = document.querySelector(`.cosmo-picker__name[for=${e.currentTarget.getAttribute('id')}]`);
    element.textContent = e.currentTarget.value;
}))

<div class="cosmo-input__group">
    <label for="filePicker" class="cosmo-label">Pick file</label>
    <div class="cosmo-input cosmo-input--picker">
        <label class="cosmo-picker__name" for="filePicker">Filename</label>
        <label class="cosmo-picker__button" for="filePicker">
            <span class="mdi mdi-upload mdi-24px"></span>
        </label>
        <input style="display: none" type="file" id="filePicker">
    </div>
</div>

Buttons

Cosmo CSS consists of only two types of buttons. The first type is a classical button just like you know it from your operating system or other frameworks. 

<button class="cosmo-button">Button</button>

Circular button

The other button type are circular buttons, they come in three sizes, small, medium and large. Medium is the default size and doesn't require any extra classes. They are best combined with an icon, just as seen in the example below. 

<button class="cosmo-circular-button cosmo-circular-button--small">
    <span class="mdi mdi-theme-light-dark"></span>
</button>
<button class="cosmo-circular-button">
    <span class="mdi mdi-theme-light-dark"></span>
</button>
<button class="cosmo-circular-button cosmo-circular-button--large">
    <span class="mdi mdi-theme-light-dark"></span>
</button>

Progress

The HTML 5 progress tag can be used to display a progress bar on the web. Currently, only determinate progress bars are supported by Cosmo CSS. 

<progress class="cosmo-progress-bar" value="13" max="100"></progress>

Indeterminate state

As of version 0.12.0 Cosmo supports the indeterminate state of progress bars.

<progress class="cosmo-progress-bar"></progress>
