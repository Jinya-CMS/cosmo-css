Master-Detail list

The Master-Detail list is a control to create a basic master detail view. At the bottom of the list you can add a button to give your users the option to create a new entry.

The Master-Detail list is a top level component and can not be put inside a tab control. Due to the fact that Cosmo CSS is a pure CSS library you need to implement the master detail logic yourself. 

<div class="cosmo-list">
    <nav class="cosmo-list__items">
        <a class="cosmo-list__item cosmo-list__item--active">Side entry 1</a>
        <a class="cosmo-list__item">Side entry 2</a>
        <a class="cosmo-list__item">Side entry 3</a>
        <button class="cosmo-button cosmo-button--full-width">Add entry to list</button>
    </nav>
    <div class="cosmo-list__content">
        <h1 class="cosmo-title">Master-Detail list</h1>
        <p>
            The Master-Detail list is a control to create a basic master detail view. At the bottom of the list you
            can add a button to give your users the option to create a new entry.
        </p>
    </div>
</div>
