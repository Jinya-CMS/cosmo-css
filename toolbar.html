Toolbar

The toolbar control can be used to group buttons into logic segments and allow the user to easier grasp what belongs to each other. 

<div class="cosmo-toolbar">
    <div class="cosmo-toolbar__group">
        <button class="cosmo-button" type="button">Toolbar button 1</button>
        <button class="cosmo-button" type="button">Toolbar button 2</button>
    </div>
    <div class="cosmo-toolbar__group">
        <button disabled="" class="cosmo-button" type="button">Toolbar button 3</button>
        <button disabled="" class="cosmo-button" type="button">Toolbar button 4</button>
    </div>
</div>
